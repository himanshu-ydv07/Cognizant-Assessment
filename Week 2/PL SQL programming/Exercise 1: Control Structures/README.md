# Exercise 1 - Control Structures in PL/SQL

## Objective

Demonstrate the use of PL/SQL control structures such as loops, conditional statements, and cursors by implementing banking-related scenarios.

## Technologies Used

* Oracle Database
* PL/SQL

## Files

* Scenario1_LoanDiscount.sql
* Scenario2_VIPPromotion.sql
* Scenario3_LoanReminder.sql

## Features

* Loop through customer records
* Apply loan interest discounts
* Promote eligible customers to VIP status
* Generate loan reminder messages

## Execution

Execute each SQL file individually using Oracle SQL Developer or SQL*Plus.
