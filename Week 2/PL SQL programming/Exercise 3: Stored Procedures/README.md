# Exercise 3 - Stored Procedures

## Objective

Demonstrate the use of PL/SQL stored procedures to automate common banking operations such as interest calculation, employee bonus updates, and fund transfers.

## Technologies Used

* Oracle Database
* PL/SQL

## Files

* ProcessMonthlyInterest.sql
* UpdateEmployeeBonus.sql
* TransferFunds.sql

## Features

* Process monthly savings account interest
* Update employee salaries with bonus percentages
* Securely transfer funds between customer accounts
* Transaction management with COMMIT and ROLLBACK

## Execution

Execute each SQL file in Oracle SQL Developer or SQL*Plus. After creating a procedure, invoke it using an anonymous PL/SQL block or the EXEC command.
